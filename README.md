# Day 1: Present Value & Cashflows (Finance Basics)

## 🔹 What is Finance?

Finance is the study of:
- Managing money over time
- Understanding risk and return

From a business perspective, finance helps us decide:
- Whether an investment is worth it
- How much something is truly worth today

---

## 🔹 Key Concept 1: Asset = Cashflows

An asset is not just a physical object.

👉 In finance, an asset is defined as a **sequence of future cashflows**:

Asset = {CF₁, CF₂, CF₃, ...}

Where:
- CF = Cashflow (money received or paid)
- Each CF happens at a different time

### Examples:
- Salary from a job
- Profits from a business
- Interest from bonds

👉 Insight: Value comes from **future money generation**

---

## 🔹 Key Concept 2: Time Value of Money

Money today is more valuable than the same amount in the future.

### Why?
- It can be invested to earn returns
- Inflation reduces future value
- Future is uncertain (risk)

---

## 🔹 Key Concept 3: Different Times = Different “Currencies”

Just like:
- ¥150 + £300 ❌ (cannot directly add)

Similarly:
- ₹100 today + ₹100 after 1 year ❌

👉 They are not directly comparable

---

## 🔹 Key Concept 4: Present Value (PV)

To compare cashflows, we convert everything to **today’s value**

PV = FV / (1 + r)^t

Where:
- PV = Present Value
- FV = Future Value
- r = interest rate
- t = time

👉 This allows fair comparison of money across time

---

## 🔹 Key Concept 5: Net Present Value (NPV)

NPV = Total present value of all cashflows

NPV helps in decision making:

- NPV > 0 → Accept investment
- NPV < 0 → Reject investment

---

## 🔹 Intuition (Most Important)

Finance is not about formulas first.

It is about thinking:
- Future money has different value
- Time affects money
- All decisions depend on comparing values today
